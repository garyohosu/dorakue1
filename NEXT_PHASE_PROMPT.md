## 指示書本文（ここからコピーして使用してください）

### 1. 目的
Phaserを用いたゲーム本編の「最小プロトタイプ（MVP）」を構築します。
タイトル画面のUI作成と、フィールド上でのキャラクターの基本移動・通行不可判定・効果音接続までを実装してください。

### 2. 実装対象

*   **タイトル画面 (TitleScene.js) の作成**
    *   画面中央に仮のゲームタイトル「暁の小径 / Path of Dawn」をテキスト表示します。
    *   メニュー項目「新しく始める」「続きから」「操作説明」「クレジット」を表示します。
    *   キーボードの上下キーまたはW/Sキーでカーソルを移動させます。移動時に [sfx.js](file:///C:/project/dorakue1/src/audio/sfx.js) の [playCursor()](file:///C:/project/dorakue1/src/audio/sfx.js#L35) を鳴らしてください。
    *   「新しく始める」を選択した際、[playConfirm()](file:///C:/project/dorakue1/src/audio/sfx.js#L54) を鳴らし、`FieldScene` に遷移します。
    *   「続きから」はセーブデータがないため、今回は選択不可（グレーアウト表示）にするか、選択時に [playCancel()](file:///C:/project/dorakue1/src/audio/sfx.js#L75) を鳴らして警告を出すだけにしてください。
*   **フィールド画面 (FieldScene.js) の作成**
    *   仮のワールドマップ（20x15 タイル程度）をPhaserのタイルマップまたはGraphics描画で用意します。
    *   アレン（主人公）の仮スプライトとして、画像ファイルの代わりにPhaserの矩形オブジェクト（例: 32x32ピクセルの赤または青の四角形）を描画します。
    *   方向キーおよびWASDキー入力で、アレンを1タイル（32px）単位で移動させます。
    *   マップデータ内に「通行不可属性」を定義します。山（茶色など）や水（青色など）のタイルにアレンが進もうとした際、移動を阻止（当たり判定）してください。
*   **音響の接続**
    *   [src/audio/sfx.js](file:///C:/project/dorakue1/src/audio/sfx.js) に実装済みの各関数を読み込み、Phaserの入力と連動させます。
    *   `FieldScene` に遷移した際、[src/audio/bgm.js](file:///C:/project/dorakue1/src/audio/bgm.js) の `playBgm('field')` を呼び出してBGMの自動ループ演奏を開始してください。

### 3. 今回実装しないもの（スコープ外）
*   敵との戦闘（BattleScene）、ランダムエンカウント
*   NPCの配置、会話メッセージウィンドウ（DialogManager）
*   武器・防具などの各種ショップ、宿屋での宿泊
*   LocalStorageへのセーブ・ロード本実装
*   ImageGenで生成予定の画像ファイルの読み込み・適用（すべて仮色・矩形で代用）
*   シーンごとの高度なBGM演出切り替え

### 4. 完了条件
1.  `npm run build` がエラーなく正常に完了すること。
2.  ブラウザ起動時、黒背景に白文字でタイトルとメニューが表示されること。
3.  メニュー選択で「移動音（カーソル音）」が鳴り、「決定音」とともにフィールド（FieldScene）へ遷移すること。
4.  フィールド上でアレン（仮の四角）を方向キーまたはWASDで上下左右に移動できること。
5.  山や水と定義したタイルにアレンが進入しようとした際、移動できないこと。
6.  `git status` と `git diff --stat` を実行し、変更差分を表示して報告すること。
